Classic Gray Shoop Theme
========================

Classic Gray is a simple Shoop Theme developed by the Shoop Team.

Static Files
-------------

Theme folder:

.. code:: sh

    shoop/themes/classic_gray/

This theme uses Bower packages and Gulp.

Gulp is required to build CSS and JavaScript files for this theme.
Run gulp in theme folder.

**Development**

.. code:: sh

    gulp watch

**Production**

.. code:: sh

    gulp --production
